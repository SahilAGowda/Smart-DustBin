Smart Dustbin

System Overview

How it works

  1. Login to access the dashboard.
  2. Upload or capture a photo of the waste item.
  3. Backend runs a CNN to detect the object.
  4. Object → Waste type mapping is applied (e.g., phone → e-waste).
  5. We estimate recyclability % and show guidance.

Waste Categories

  • Plastic
  • Metal
  • Paper
  • Glass
  • Organic
  • E-waste
  • Medical
  • Other

Note: This is a prototype skeleton. Recyclability % is a heuristic. For production, integrate a real model, local regulations, and proper safety guidance.
